Quick Reference

SQL Cheat Snippets

Copy-paste ready patterns for the most common SQL interview scenarios.

Window Function
SELECT name, salary, RANK() OVER ( PARTITION BY dept ORDER BY salary DESC ) AS rnk FROM employees;
CTE + Join Pattern
WITH top_users AS ( SELECT user_id, COUNT(*) orders FROM orders GROUP BY user_id ) SELECT u.name, t.orders FROM users u JOIN top_users t ON u.id = t.user_id;
7-Day Moving Average
SELECT date, value, AVG(value) OVER ( ORDER BY date ROWS BETWEEN 6 PRECEDING AND CURRENT ROW ) AS ma_7day FROM timeseries;
Data Quality Check
-- NULLs + duplicates SELECT COUNT(*) total, COUNT(email) non_null, COUNT(DISTINCT email) uniq FROM users; SELECT email, COUNT(*) cnt FROM users GROUP BY email HAVING cnt > 1;
